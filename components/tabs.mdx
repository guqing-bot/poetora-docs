    class PoetoraClient {
      constructor(apiKey) {

        });

      }
    }

    ```python
    # Poetora API 客户端 - Python 版本
    import requests
    import json

    class PoetoraClient:
        def __init__(self, api_key):
            self.api_key = api_key
            self.base_url = 'https://api.poetora.com/v1'
            self.headers = {
                'Authorization': f'Bearer {api_key}',
                'Content-Type': 'application/json'
            }

        def create_document(self, data):
            """创建新文档"""
            response = requests.post(
                f'{self.base_url}/documents',
                headers=self.headers,
                json=data
            )
            return response.json()

        def get_document(self, doc_id):
            """获取文档详情"""
            response = requests.get(
                f'{self.base_url}/documents/{doc_id}',
                headers=self.headers
            )
            return response.json()

    # 使用示例
    client = PoetoraClient('your-api-key')
    doc = client.create_document({
        'title': '新文档',
        'content': '这是文档内容'
    })
    print(f"文档ID: {doc['id']}")
    ```

    ```php
    <?php
    // Poetora API 客户端 - PHP 版本
    class PoetoraClient {
        private $apiKey;
        private $baseURL;

        public function __construct($apiKey) {
            $this->apiKey = $apiKey;
            $this->baseURL = 'https://api.poetora.com/v1';
        }

        public function createDocument($data) {
            $curl = curl_init();

            curl_setopt_array($curl, [
                CURLOPT_URL => $this->baseURL . '/documents',
                CURLOPT_RETURNTRANSFER => true,
                CURLOPT_POST => true,
                CURLOPT_POSTFIELDS => json_encode($data),
                CURLOPT_HTTPHEADER => [
                    'Authorization: Bearer ' . $this->apiKey,
                    'Content-Type: application/json'
                ]
            ]);

            $response = curl_exec($curl);
            curl_close($curl);

            return json_decode($response, true);
        }

        public function getDocument($id) {
            $curl = curl_init();

            curl_setopt_array($curl, [
                CURLOPT_URL => $this->baseURL . '/documents/' . $id,
                CURLOPT_RETURNTRANSFER => true,
                CURLOPT_HTTPHEADER => [
                    'Authorization: Bearer ' . $this->apiKey
                ]
            ]);

            $response = curl_exec($curl);
            curl_close($curl);

            return json_decode($response, true);
        }
    }

    // 使用示例
    $client = new PoetoraClient('your-api-key');
    $doc = $client->createDocument([
        'title' => '新文档',
        'content' => '这是文档内容'
    ]);
    echo "文档ID: " . $doc['id'];
    ?>
    ```

## 组件属性

### Tabs 属性

| 属性名        | 类型                         | 必需 | 默认值         | 描述                 |
| ------------- | ---------------------------- | ---- | -------------- | -------------------- |
| `defaultTab`  | `number`                     | ❌   | `0`            | 默认激活的标签页索引 |
| `orientation` | `'horizontal' \| 'vertical'` | ❌   | `'horizontal'` | 标签页方向           |

### Tab 属性

| 属性名     | 类型      | 必需 | 默认值  | 描述             |
| ---------- | --------- | ---- | ------- | ---------------- |
| `title`    | `string`  | ✅   | -       | 标签页标题       |
| `icon`     | `string`  | ❌   | -       | 标签页图标       |
| `disabled` | `boolean` | ❌   | `false` | 是否禁用该标签页 |

## 最佳实践

### 标签组织

- **逻辑分组** - 将相关内容组织在一起
- **标题简洁** - 使用简短而描述性的标题
- **数量适中** - 避免标签页过多影响用户体验
- **顺序合理** - 按重要性或使用频率排序

### 内容设计

- **独立完整** - 每个标签页的内容应该相对独立
- **长度均衡** - 避免某个标签页内容过长或过短
- **视觉一致** - 保持各标签页的视觉风格一致
- **交互友好** - 考虑移动设备的使用体验

### 性能优化

- **懒加载** - 对大量内容实施懒加载
- **图片优化** - 压缩标签页中的图片
- **代码分割** - 将复杂内容分割到不同标签页
- **缓存策略** - 合理缓存标签页内容

## 可访问性

标签页组件支持完整的可访问性功能：

- **键盘导航** - 支持 Tab、Arrow 键导航
- **屏幕阅读器** - 提供适当的 ARIA 标签
- **焦点管理** - 清晰的焦点指示
- **状态提示** - 明确指示当前激活的标签页

### 键盘操作

| 按键              | 功能                      |
| ----------------- | ------------------------- |
| `Tab`             | 进入/离开标签页组件       |
| `←` / `→`         | 在标签页之间切换          |
| `Space` / `Enter` | 激活当前聚焦的标签页      |
| `Home` / `End`    | 跳到第一个/最后一个标签页 |
