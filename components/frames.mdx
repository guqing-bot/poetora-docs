# 图片框

图片框组件用于包装图片、视频或其他视觉内容，提供一致的样式和可选的标题说明。它能让你的视觉内容更加专业和统一。

## 组件属性

| 属性名    | 类型      | 必需 | 默认值 | 描述                 |
| --------- | --------- | ---- | ------ | -------------------- |
| `caption` | `string`  | ❌   | -      | 图片或内容的标题说明 |
| `border`  | `boolean` | ❌   | `true` | 是否显示边框         |
| `shadow`  | `boolean` | ❌   | `true` | 是否显示阴影效果     |
| `rounded` | `boolean` | ❌   | `true` | 是否使用圆角样式     |

## 最佳实践

### 图片选择

- **高质量图片** - 使用清晰、高分辨率的图片
- **相关性强** - 确保图片与内容主题相关
- **版权清晰** - 使用有授权的图片素材
- **文件优化** - 压缩图片大小以提升加载速度

### 标题编写

- **描述准确** - 准确描述图片内容和用途
- **简洁明了** - 避免过长的标题文字
- **信息丰富** - 提供必要的上下文信息
- **一致风格** - 保持标题的格式和语调一致

### 布局考虑

- **空间平衡** - 合理安排图片和文字的比例
- **视觉流程** - 考虑用户的阅读顺序
- **响应式设计** - 确保在不同设备上的良好显示
- **加载性能** - 优化图片加载策略

### 可访问性

- **替代文本** - 为所有图片提供有意义的 alt 文本
- **对比度** - 确保标题文字的对比度足够
- **键盘访问** - 支持键盘用户的导航
- **屏幕阅读器** - 提供清晰的内容结构

## 技术细节

### 图片优化

```html
<!-- 响应式图片 -->
<frame caption="响应式图片示例">
  <img
    src="/image-800.jpg"
    srcset="
      /image-400.jpg   400w,
      /image-800.jpg   800w,
      /image-1200.jpg 1200w
    "
    sizes="(max-width: 768px) 400px, (max-width: 1024px) 800px, 1200px"
    alt="响应式图片"
  />
</frame>
```

### 懒加载

```html
<!-- 懒加载图片 -->
<frame caption="懒加载图片示例">
  <img
    src="/placeholder.jpg"
    data-src="/actual-image.jpg"
    loading="lazy"
    alt="懒加载图片"
  />
</frame>
```

### 现代图片格式

```html
<!-- 使用 WebP 格式 -->
<frame caption="现代图片格式">
  <picture>
    <source
      srcset="/image.webp"
      type="image/webp"
    />
    <source
      srcset="/image.jpg"
      type="image/jpeg"
    />
    <img
      src="/image.jpg"
      alt="现代图片格式示例"
    />
  </picture>
</frame>
```
