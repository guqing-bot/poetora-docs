## 基础用法

### 简单步骤

创建基本的步骤指南：

<Steps>
  <Step title="注册账号">
    访问 Poetora 官网，点击"立即注册"按钮开始创建你的账号。

    需要提供：
    - 有效的邮箱地址
    - 安全的密码（至少8位字符）
    - 用户名或真实姓名

  </Step>
  <Step title="验证邮箱">
    检查你的邮箱，点击验证邮件中的确认链接完成账号激活。

    如果没有收到邮件，请：
    - 检查垃圾邮件文件夹
    - 确认邮箱地址输入正确
    - 等待几分钟后重新发送

  </Step>
  <Step title="创建工作空间">
    登录后创建你的第一个工作空间，设置项目名称和基本信息。

    工作空间设置包括：
    - 工作空间名称
    - 团队成员邀请
    - 权限配置
    - 主题选择

  </Step>
</Steps>

    ```javascript
    import { PoetoraClient } from '@poetora/sdk';

    const client = new PoetoraClient({
      apiKey: process.env.POETORA_API_KEY,
      baseURL: 'https://api.poetora.com/v1'
    ```

    <Warning>
    请确保将 API 密钥存储在环境变量中，不要直接写在代码里。

    ```javascript
    const editor = await client.createEditor({
    ```

  <Step title="基础功能实现" icon="code">
    💻 **实现核心功能**

    **文档操作接口：**
    ```typescript
    // src/services/document.ts
    import { poetora } from '../lib/poetora';

    export class DocumentService {
      async createDocument(title: string, content?: string) {
        return await poetora.documents.create({
          title,
          content: content || '',
          format: 'mdx'
        });
      }

      async getDocument(id: string) {
        return await poetora.documents.get(id);
      }

      async updateDocument(id: string, updates: any) {
        return await poetora.documents.update(id, updates);
      }

      async deleteDocument(id: string) {
        return await poetora.documents.delete(id);
      }

      async listDocuments(options?: any) {
        return await poetora.documents.list(options);
      }
    }
    ```

    **错误处理：**
    ```typescript
    // src/utils/error-handler.ts
    export class PoetoraError extends Error {
      constructor(
        message: string,
        public statusCode?: number,
        public details?: any
      ) {
        super(message);
        this.name = 'PoetoraError';
      }
    }

    export function handlePoetoraError(error: any) {
      if (error.response) {
        throw new PoetoraError(
          error.response.data.message,
          error.response.status,
          error.response.data
        );
      }
      throw new PoetoraError(error.message);
    }
    ```

  <Step title="前端集成" icon="monitor">
    🖥️ **集成到前端应用**

    **React 组件示例：**
    ```tsx
    // src/components/DocumentEditor.tsx
    import React, { useEffect, useState } from 'react';
    import { PoetoraEditor } from '@poetora/react';
    import { DocumentService } from '../services/document';

    interface Props {
      documentId?: string;
    }

    export function DocumentEditor({ documentId }: Props) {
      const [document, setDocument] = useState(null);
      const documentService = new DocumentService();

      useEffect(() => {
        if (documentId) {
          loadDocument();
        }
      }, [documentId]);

      const loadDocument = async () => {
        try {
          const doc = await documentService.getDocument(documentId!);
          setDocument(doc);
        } catch (error) {
          console.error('Failed to load document:', error);
        }
      };

      const handleSave = async (content: string) => {
        try {
          await documentService.updateDocument(documentId!, { content });
          console.log('Document saved successfully');
        } catch (error) {
          console.error('Failed to save document:', error);
        }
      };

      return (
        <div className="editor-container">
          <PoetoraEditor
            document={document}
            onSave={handleSave}
            autoSave={true}
            saveInterval={5000}
          />
        </div>
      );
    }
    ```
